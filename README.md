CarService - Система управления автосервисом

Стек технологий
Backend:
1)ASP.NET Core 8.0 - основной фреймворк
2)Entity Framework Core 8.0 - ORM для работы с базой данных
3)MySQL - система управления базами данных
4)ASP.NET Core Identity - система аутентификации и авторизации

Frontend:
1)Razor Pages - технология представлений
2)Bootstrap - CSS-фреймворк
3)HTML/CSS/JavaScript - клиентская сторона

Архитектура:
1)MVC Pattern - архитектурный шаблон
2)Repository Pattern - шаблон доступа к данным
3)Dependency Injection - встроенная система зависимостей

Как пользоваться:
1)Регистрация и вход
  1.Зарегистрируйтесь как пользователь
  2.Администратор создается автоматически

2)Просмотр услуг
  1.На главной странице просматривайте доступные услуги
  2.У каждой услуги есть название, цена и изображение
3)Запись на услугу
  1.Выберите услугу
  2.Заполните форму с вашими данными и информацией об автомобиле
  3.Система автоматически создаст клиента, автомобиль и заказ
4)Админ-панель (только для администраторов)
  1.Управление услугами: добавление, редактирование, удаление
  2.Управление клиентами: просмотр, редактирование, удаление
  3.Просмотр всех заказов
5)Управление клиентами
  1.Просмотр списка всех клиентов
  2.Редактирование данных клиентов и их автомобилей
  3.Удаление клиентов с каскадным удалением связанных данных
